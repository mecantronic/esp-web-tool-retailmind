Instalador de Firmware – RetailMind

Instrucciones

  1. Conectá tu dispositivo ESP32 por USB a la computadora.
  2. Usá el navegador Chrome o Edge (otros no son compatibles).
  3. Presioná el botón Instalar Firmware correspondiente al modelo.
  4. Seleccioná el puerto USB cuando el navegador lo solicite.
  5. Esperá a que el proceso termine sin desconectar el dispositivo.
  6. ¡Listo! El firmware estará flasheado correctamente.
